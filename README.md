## アプリ概要
- 行きたい場所を保存しておく
- 行きたい場所に訪問したら、その場所を訪問した場所に登録し、行きたい場所からは削除
- 訪問した場所をレビューと点数を付けて保存する
- 行きたい場所に登録してない場所も、訪問した場所に登録できる
- 訪問した場所でもう一度行きたい場所は、もう一度行きたい場所として保存する
- アプリのイメージは、todoアプリのタスクを管理するのではなく、行きたい場所,訪問済みの場所を管理するアプリのイメージです。
